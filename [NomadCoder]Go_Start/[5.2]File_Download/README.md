
## File Encoding Error --> 다운받은 csv 파일 한글 깨짐 현상

### 해결방안(1)
1. 다운받은 csv 파일 우클릭 
2. 메모장으로 열기
3. 저장

### 해결방안(2)
1. 엑셀 빈 파일을 열고 
2. 데이터 클릭
3. 새쿼리 클릭 
4. 파일에서 클릭 
5. csv에서 클릭 
6. jobs.csv 선택
7. 가져오기 클릭
8. 파일원본 드롭다운 버튼 클릭 후 65001:유니코드(UTF-8) 선택 
9. 로드 클릭
10. 디자인 탭 선택
11. 머리글 행 체크 해제
12. 첫 번째 행 삭제 
13. 표스타일 : 없음 선택
14. 저장
